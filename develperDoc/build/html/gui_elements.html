class Windows.LineEdit.LineEdit¶
Class

Subclasses QLineEdit to add additional functionality

__hasDoubleValidator = False¶
Attr

True if DoubleValidator has been set.

__hasIntValidator = False¶
Attr

True if IntValidator has been set.

__hasRegExpValidator = False¶
Attr

True if RegExpValidator has been set.

checkContent()¶
Classmethod

Does check the content of a line edit with the help of the applied validator. Depending on the check result the background of the line edit is changed.

enableDoubleValidator(bottom, top, decimals=0)¶
Classmethod

Creates an DoubleValidator and sets it for the current line edit.

Parameters

  • bottom – lower value of validation border

  • top – upper value of validation border

  • decimals – number of decimals to be checked

enableIntValidator(bottom, top)¶
Classmethod

Creates an IntValidator and sets it for the current line edit.

Parameters

  • bottom – lower value of validation border

  • top – upper value of validation border

enableRegExpValidator(regexp)¶

Creates an RegExpValidator and sets it to the current line edit. :param regexp: the RegExp to be applied to the validator.

eventFilter(source, event)¶
Classmethod

Catches specific events and controls (the updates of the help window and the validation of the user inputs).

setHelpBar(helpBar)¶
Classmethod

Configure the help bar of a specific window where the user help text shall be displayed during program execution.

Parameters

helpBar – Instance of the respecitive help bar to work with

setHelpText(helpText)¶
Classmethod

Herein you set the help text for each LineEdit which shall be displayed if the mouse pointer is located above the LineEdit or during data edit.

Parameters

helpText – Help text to be displayed

setText(*args, **kwargs)¶
Classmethod

Calls first the super class to set the text and triggers afterwards the value check making sure the background is set according to the verification result.
