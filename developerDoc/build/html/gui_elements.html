class gui.elements.WindowBtnBar.WindowBtnBar(buttons=15)¶
Class

Displays the standard buttons under each data window.

Button Layout:

[Apply] [Ok] [Cancel]
             [Help]

Bitmask controlling which buttons will be displayed:

Apply    0b1000
Ok       0b0100
Cancel   0b0010
Help     0b0001

Attr

The user help page which will be opened if not an individual file was configured
