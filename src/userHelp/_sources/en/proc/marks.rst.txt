***********************

All details you will find on Peres website |pere_link|.

	*************************************************************
   	*           7. MARKS
   	*************************************************************
   	25.	0.5	0.15
	
Description
***********

Header::

	*************************************************************
  	*           7. MARKS
  	************************************************************* 
   
The Header will be created automatically by lepg.

3 Parameters define the sewing marcage. 

Marks spacing::

	25.

Point radius::

	0.5
	
Point displacement::

	0.15
