Sort by Rib NumberÂ¶

The button Sort by Rib Number can be used to rearrange the definition lines. If for whatever reasons you will rearrange the lines, just define the rib numbering in an ascending order and press the Order button afterwards. Lepg will reorder the lines according to the numbering youâve choosen.

ATTENTION you have to order the lines in a way that the wing is defined from the middle to the tip.

A more detailed description you can find here Laboratori d'envol website.
